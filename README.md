# 📖 Hello Rails React <a name="about-project"></a>



**Hello Rails React** is a cutting-edge web application that seamlessly integrates React.js and Rails to provide an immersive development experience. This project serves as an exploration into the powerful synergy between these two technologies.

By leveraging the power of webpack and Rails, Hello Rails React combines the declarative and component-based nature of React with the robustness and scalability of Rails. Developers can harness the efficiency of React's virtual DOM and reusable components while benefiting from Rails' convention-over-configuration philosophy and extensive ecosystem.

With Hello Rails React, you can effortlessly build interactive user interfaces, manage complex state, and handle data flow efficiently. The seamless integration between React and Rails enables rapid development and easy maintenance, making it an ideal choice for ambitious projects.

Whether you are a seasoned developer or an enthusiast, Hello Rails React opens up a world of possibilities for building modern, performant, and feature-rich web applications. Experience the best of both worlds with this innovative React and Rails integration.

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:
- Browser
- IDE
- PostgreSQL database installed
- Ruby on Rails
- Ruby


### Setup
Clone the repository to desired directory on local computer.
```
   git clone https://github.com/wacoo/hello-rails-react

   cd hello-rails-react
```

### Install
To install the project please run this command:
```
./bin/bundle install
```
### Usage
To start the project please run this commands:
```
rails db:create db:migrate
./bin/dev
```
